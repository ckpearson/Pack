Welcome to Pack

Pack is a super-simple app that lets you share relatively small files with other people by using images as the transport medium.

Many times (take GitHub for example!) you want to share a simple file, say a word document, a spreadsheet or other such files, but the system you're using doesn't accommodate file uploads, Pack lets you share the data using a convenient image instead!

Pack takes the file you want to share, compresses it a bit and then produces an image that contains that data; to get the data out again someone just needs to pass the image back through Pack and out comes the file (filename too!)
